# AI Search Readiness Checker

A WordPress plugin to evaluate your site's readiness for AI search engines (Perplexity, Gemini, ChatGPT Search).

## Features
- Content structure analysis (Headings).
- Schema markup detection.
- E-E-A-T signal verification.
- robots.txt and llms.txt checks.
- XML Sitemap status.

## Installation
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate via the WordPress admin.
3. Go to **Tools > AI Search Readiness** to see your report.

## Why this exists
Based on the February 2026 guidelines from WordPress.com, AI search optimization requires a shift from keyword density to extraction-friendly structures and clear ownership signals.
