Reviews your WordPress site's readiness for AI-powered search engines based on 9 key steps.

== Description ==

AI Search Readiness Checker audits your WordPress site against 9 key factors that determine how well AI-powered search engines can discover, understand, and surface your content.

== Installation ==

1. Upload the `ai-search-readiness-checker` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Tools > AI Search Readiness to run the audit.

== Frequently Asked Questions ==

= What AI search engines does this check for? =

The checker evaluates general readiness factors applicable to all AI-powered search platforms.
